## Animation Production Techniques: Motion Without Redrawing

**Time:** 1 Week (4–5 Class Periods)

**Due Date:** Following Monday before class

---

# Learning Objectives

By the end of this assignment students will:

* Use transform tools effectively
* Animate through position, scale, rotation, and opacity
* Create smooth motion with minimal drawing
* Understand keyframe-based workflows

---

# Production Question

## How can an animator create motion without drawing new frames?

Many professional animations rely heavily on transforms and keyframes.

Transform animation allows artists to create motion while reusing existing artwork.

---

# Part 1 – Analysis (Day 1)

Study examples of:

* Motion graphics
* UI animations
* Logo reveals
* Floating objects

### Observation Questions

1. Which objects are actually being redrawn?
2. Which objects are simply transformed?
3. How does timing affect motion quality?

# Part 3 – Animation Production (Days 2–4)

### Requirements

* 5–10 second animation
* Minimum 3 animated objects
* Use:

  * Position
  * Scale
  * Rotation
  * Opacity
* Maximum 5 hand-drawn frames

# Part 4 – Reflection (Day 5)

1. How many drawings were created?
2. How many transformations were used?
3. What work was saved through transforms?
4. Which transform was most effective?
